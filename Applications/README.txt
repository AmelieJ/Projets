##################################### MANUEL D'INSTALLATION #####################################


***********Fichiers mis à disposition************************************************************

*	- Gnews.jar : executable jar					    			*

*	- test.sql : base de donnée à importer sous PhpMyAdmin depuis Wampserver2.4-x86		*

********************************************************************************************
- Le JRE est indispensable pour exécuter une application java, si nécessaire le télécharger à l'adresse suivante : 										http://www.java.com/fr/download/


- Installation de Wampserver2.4-x86

- Lancer Wampserver2.4-x86

- En bas à droite du bureau, clic gauche sur Wampserver et cliquer sur phpMyAdmin

- Se loguer en tant que : login = "root" mdp = "" et cliquer sur Executer

- Cliquer sur la base "creation_table" à gauche sous l'option "Tables récentes"

- CLiquer sur l'onglet "Importer", choisir le fichier test.sql et cliquer sur Executer

- Le paramètrage de l'application est terminé, lancer l'executable Gnews.jar

- Se fier à la documentation d'installation pour utiliser l'application
